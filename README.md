# devops-exam

## Description

JS APP

### Prerequisites

-   [Node JS / NPM](https://nodejs.org/en/)

### Installing

Install dependencies

```
npm i
```

### Building

Building the app

```
npm run build
```

### Start

Serving the solution at http://localhost:8080/ (default one)

```
npm start
```

### Notes

To see the changes applied once the app is served, don't forget to build first
